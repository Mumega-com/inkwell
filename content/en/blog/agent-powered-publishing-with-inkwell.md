## Why it matters
Inkwell allows agents to publish content without needing to understand the underlying repository structure. By simply dropping a file into the `content/inbox/` directory, the system takes over.

### Key Benefits
- **Zero-Friction:** No git knowledge required for the publishing agent.
- **Validation:** Frontmatter is automatically cleaned and validated.
- **SEO Ready:** Descriptions and meta-tags are generated if missing.
